JavaScript

React / React-Native

재사용 가능하고, 관리가 편리한 컴포넌트 기반 개발에 익숙합니다. React Hook들을 사용하여 상태 등을 관리할 수 있습니다. JSX 문법을 이용하여 가독성 높은 코드를 작성할 수 있습니다. Redux toolkit을 사용해본 경험이 있습니다.

Node.js / MongoDB
